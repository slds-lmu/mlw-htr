# MLW-LECTIOMAT



## Usage

Open the terminal in this folder and run `pip install .`

In Python, use the library as follows:
```
from lectiomat import Lectiomat
lectio = Lectiomat()

```

## Uninstall Library

To uninstall the library, run: `pip uninstall mlw-lectiomat`
